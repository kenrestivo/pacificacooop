How to print these pages:
  • Before printing, turn off the headers and footers in your browser. Choose 'Page Setup' from your 'File' menu, and in the 'Margins & Header/Footer' section, clear out all the junk in 'Headers & Footers'-- set it to blank.

    Sample Page Setup box for Firefox, correctly configured.
  • You can also reduce the size of the margins in 'Page Setup', if you want try to squeeze more stuff on one page.
  • Choose 'Page Preview' from your File menu to see what it'll look like printed.
  • The footer will automatically appear at the bottom of each page when printed.
  • If you scroll down here, the sheets will probably look awful on screen. Don't panic; instead use Page Preview instead to see how they print.
  • Put paper in your printer and print away!
  • If you run into trouble using Explorer, try with Firefox
  • This page and everything in it is just an HTML template which can be modified by any competent web designer you want it changed. Just ask.
